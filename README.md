# Aplikasi Kasir Cafe (UAS)

Aplikasi kasir cafe berbasis desktop menggunakan Windows Forms
yang dibuat untuk memenuhi tugas UAS.

## Informasi Project
- Nama Project : WindowsApp1
- Platform     : Desktop
- Bahasa       : C#
- Database     : MySQL
- IDE          : Visual Studio 2022

## Fitur Aplikasi
- Manajemen Produk
- Transaksi Penjualan
- Perhitungan Total dan Kembalian
- Penyimpanan Data ke Database MySQL

## Struktur Folder
- docs     : Dokumen UAS
- src      : Source code aplikasi
- database : File SQL database

## Cara Menjalankan Aplikasi
1. Import database dari folder `database/kasir_cafe.sql`
2. Buka file `WindowsApp1.sln` menggunakan Visual Studio 2022
3. Jalankan aplikasi
